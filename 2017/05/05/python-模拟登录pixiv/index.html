需要的工具及运行环境

  • python 3.6
  • requests 库
  • re 库
  • chrome 浏览器

分析URL

进入pixiv官网,
按F12 开打开发者工具界面，点击官网上的LOGIN 按钮,
跳转到登录界面,
开发者工具界面依次选择“Network”->“Doc”，左侧Name栏有请求信息。

如图可知三个重要信息：

  • 请求方式:GET
  • User-Agent
  • get的请求参数

尝试登录

开发者界面，勾选Preserve log。

输入账号，密码登录pixiv
开发者工具界面点击“XHR” ，左侧name栏选择“login?lang=en”

获得信息：

  • 发送表单方法post
  • 表单中要发送的数据

构建要发送数据的字典

注意到字典中有一个key 为post_key，这个东西你每次登录之前会给你个唯一值，所以我们要获得这个值，并将赋值给post_key的value。
